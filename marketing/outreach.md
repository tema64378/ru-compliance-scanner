# Холодный аутрич

> Принцип: не пугаем враньём, а реально помогаем. Сначала польза (бесплатный отчёт),
> потом оффер. Все суммы — настоящие (КоАП, ред. 30.05.2025).

## Как находить лиды
1. Берёшь список сайтов (местный бизнес, интернет-магазины, веб-студии, инфобизнес).
2. Прогоняешь сканером:
   ```bash
   python cli.py сайт1.ру сайт2.ру сайт3.ру --csv leads.csv
   ```
3. Берёшь из `leads.csv` тех, у кого нарушений > 0, и пишешь им.

## Важно (чтобы не было спама и проблем)
- Пиши по делу, с конкретикой по ИХ сайту (не массовая рассылка-копипаста).
- Не обещай «гарантию от штрафа» — обещай снижение риска и готовые документы.
- B2B-рассылка по email требует аккуратности (тот же 152-ФЗ 😄) — лучше точечно и персонально.
